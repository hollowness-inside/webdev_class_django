## Реализованный функционал

### Маршруты приложения

| Маршрут | Метод | Описание |
|---------|-------|---------|
| `/` | GET | Главная страница (Function-Based View) |
| `/about/` | GET | Страница "О нас" (Function-Based View) |
| `/student/<int:student_id>/` | GET | Информация о студенте (Class-Based View) |
| `/course/<slug:course_slug>/` | GET | Информация о курсе (Class-Based View) |

### Специальные обработчики

- Пользовательская страница ошибки 404 (не найдено)
- Пользовательская страница ошибки 500 (ошибка сервера)
